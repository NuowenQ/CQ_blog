Application Description

Problem Statement

If there is an invention is implemented, there must be steps and
procedures to follow and comply with. Similar to the existing
self-checkout station, there are standard procedures to follow up on
before they can take the purchased items and leave the supermarket.
However, people can make mistakes while following the steps in which
they don’t have an alternating channel to return to the previous step
which may confuse the customers. Also, customers need to place the
scanned items onto a platform before they can scan the next item, which
sometimes can cause an error in this step as the platform is not so
sensitive to detect whether the scanned item has been put onto it or not
sometimes. All these minor errors can cause huge confusion for the users
which is not user-friendly enough.

Project Description

Based on the problem statement described above, we came out with the
idea of a machine that works with a conveyor belt that can identify the
purchased items using a camera programmed to do machine vision scanning.
The conveyor belt will deliver the purchased items to the scanned area
and the camera will capture the items and show the price on the display
to show the customers how much they need to pay.

Marketing Plan

The targeted users are the existing supermarket. There are around 23
famous supermarket brands in Singapore, including NTUC FairPrice, Sheng
Siong and Cold Storage, and convenience stores and local shops. In total
round up to a few thousand of such stores in Singapore. Our project will
significantly benefit them by simplifying the customer self-checkout
process and making payments. This directly solves the problem of people
waiting to complete the payment, resulting in a long and troublesome
queue. 

This project aims not only to provide these target users with a machine
that automatically detects the item and calculates the price. In order
to identify the net weight difference for a similar product, a more
precise model is needed to be trained in order to detect the difference.
Therefore, we can provide the target users data training services with a
certain amount of subscription fee required. Our data training service
aims to provide users with the most up-to-date and accurate data so it
will be convenient for supermarkets to adopt such technology.

Penetration rate: The penetration rate shows the extent to which the
target users have adopted the product. We would like to achieve a
penetration rate of 50%.  Using the formula: Penetration rate = Number
of adopters/ Total market size * 100 We have calculated that we should
expect a number of adopters. From this formula, we have calculated
that in order to obtain a penetration rate of  50%, a number of 1500
adopters are expected.

Challenges

There are several problems that we have encountered during the process.

Firstly, we have encountered problems regarding the design problem of
the prototype.

One of the design problems is that we found it very difficult for the roller
and belt to work together due to the lack of friction in between. So, we
have changed to wrap the roller with a bunch of rubber bands to increase
the friction to ensure the roller can push the belt forward.

The other design problem is that connect the DC motor with the roller.
The original design is to connect the roller with the DC motor using hot
glue, but we found it is not very efficient as the connection point will
loosen after a few rounds of testing. The changes we have made are
connecting the wheels attached with satay sticks to ensure the conveyor
belt can run completely smooth as the satay sticks are perfectly matched
with the fluting medium of the cupboard.

The second technical problem is that we found it very difficult to find
a suitable computer vision model to train. Thus, it is very
time-consuming as we must keep trying different models until we find the
most suitable model.

Lastly, it is very tedious to train the model and set up the process
whenever we want to do the testing. Training the data including taking
pictures of the items in different orientations and postures and doing
data labelling by setting rectangular regions over the label items.

Creativity and Innovation

Creativity

Innovation

Machine vision:

One of the innovative ideas in solving the issue we are working on is
that we make use of one of the machine learning technologies - computer
vision provided by the TensorFlow lite python API. To detect the name of
the items and calculate the price.

TensorFlow Lite supports a variety of object detection models, for
example,e the region-based framework (R-FCN).

TensorFlow lite is a lightweight version of TensorFlow that is designed
for mobile and embedded devices. It enables on-device machine learning
inference with low latency and high performance. Therefore, we feel it
is suitable to be implemented on the Raspberry Pi operating system.

Generally, the object detection task is carried out in three steps:

  • Generates the small segments in the input as shown in the image below.
    As you can see, the large set of bounding boxes are crossing the full
    image.

  • Feature extraction with CNN is carried out for each segmented
    rectangular area to predict whether the rectangle contains a valid
    object.

  • Overlapping boxes are combined into a single bounding rectangle
    (Non-Maximum Suppression) now a cat is detected and localised.

Here is a clear explanation of how we have modified and developed such
technology and apply it to solve our problem:

  1. Object labelling: In order to let the computer recognize the
    items which are within our concerns, the first step is to take
    pictures and label the objects in the pictures in different
    positions and lighting conditions. For this step, I used “labelling”
    to label the objects.

  2. Model conversion: The object detection model is trained using
    TensorFlow and then converted to TensorFlow Lite format. This
    conversion process optimises the model for mobile devices and
    reduces the file size, making it easier to deploy. After labelling
    the pictures, I then use this google collab published by the company
    TensorFlow to generate the model.

  3. Input preprocessing: The input image is resized and normalised
    before being fed into the model. The normalisation process helps to
    improve the accuracy of the model.

  4. Inference: The TensorFlow Lite interpreter runs the optimized
    model on the input image and generates the output, which includes
    the bounding boxes and class scores for each detected object.

  5. Postprocessing: The output from the model is post processed to
    filter out low-confidence detections and perform non-maximum
    suppression to remove overlapping bounding boxes.

  6. Display: The final results, including the bounding boxes and
    class labels, are displayed on the screen.

TensorFlow Lite offers a high-level API for object detection that makes
it easy to run object detection models on mobile devices. It also
provides low-level APIs for advanced users who want to customize their
object detection pipeline.

Object detection is one of the important applications of computer vision
and is used in many real-world scenarios. TensorFlow Lite supports a
variety of object detection models, including single-shot detection
(SSD) and region-based fully convolutional networks (R-CNN).

Conclusion

In conclusion, installing an automated checkout system in supermarkets
can have a number of positive effects on both the clientele and the
company. This cutting-edge solution can improve the checkout process,
saving customers important time and delivering a faster, more convenient
shopping experience in response to the growing need for efficiency and
convenience. On the business side, the automation of checkout procedures
can result in shorter client wait times and higher customer satisfaction
levels. Furthermore, the inclusion of cutting-edge technology in
checkout machines can deliver useful information and insights that can
be used to optimize processes and enhance the shopping experience as a
whole. Overall, the automated checkout device is a wise investment that
can help the business and its clients greatly.
